¡Preguntas frecuentes!

Siempre estamos dispuestos a resolver tus dudas. ;-)

  • ¿Cómo funciona el sistema de registro por número telefónico?

Para iniciar sesión o registrarse mediante un número telefónico se debe consultar la pestaña 'Iniciar con teléfono'. Después, se deben introducir nombre completo, de usuario, el número y resolver el captcha. Una vez hecho esto, se enviará un código de verificación, a través del cual se habilitará el inicio de sesión.

Si el número telefónico ya estaba registrado en el sistema, entonces los datos originales del usuario permanecerán intactos, lo que quiere decir que los campos de nombre completo y de usuario resultan, en este caso, impertinentes.

  • ¿Por qué el sistema no valida que los correos electrónicos registrados sean válidos?

Esto se debe a que el sistema de autenticación de usuarios utiliza el correo como un identificador, lo que significa que no se empleará para mecanismos de correo electrónico.

  • ¿No puedo modificar mis datos de usuario?

Por desgracia, esto no resulta posible por el momento. Debido al poco tiempo disponible para el desarrollo del proyecto, la implementación de funciones adicionales demorará más de lo planeado.

  • ¿Cómo puedo agregar o modificar alguna receta del sistema?

Solo el usuario administrador puede hacer esto, así que si se necesita realizar alguna operación con recetas, entonces se debe contactar con él.

  • ¿Por qué algunas operaciones llevan múltiples segundos en completarse?

Varias consultas que el sistema hace con la base de datos son de carácter asíncrono, debido a lo cual es probable que, sin posponer la ejecución general de la página, ocurran eventos inconsistentes en el mismo.

  • ¿Qué uso le puedo dar al código QR de una receta?

Cada uno de los códigos QR contiene toda la información de la receta que le corresponde, así que puede emplearse para almacenar y compartir digitalmente las instrucciones de preparación de algún platillo de interés.

  • ¿A dónde se envían los mensajes de la pestaña de contacto?

Al correo electrónico del administrador, aunque se utiliza como intermediario el correo electrónico de la aplicación web, debido a lo cual no será posible recibir una respuesta directa.

  • ¿Qué puedo hacer si encuentro alguna falla en el sistema?

Tratar de recopilar todas las circunstancias bajo las cuales el error ocurre y reportarlo a través de la pestaña de contacto o un mensaje directo vía correo electrónico.

  • ¿No puedo restablecer mi contraseña en caso de haberla olvidado?

Por el momento, no se puede hacer esto. Sin embargo, queremos enfatizar en que estamos trabajando continuamente para que el sistema reciba actualizaciones constantes, así que ese problema se solucionará pronto.
